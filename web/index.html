That is, it displays the steps used during the conversion process to help the user better understand the conversion process.

BlackJack Casino Simulation

It is a stand-alone java program that simulates a BlackJack casino. There are two modes in the simulation, a manager and player mode. The manager mode is the simulation, whereas, in the player mode the user plays the game. Each table in the Casino is its own thread. The user is also able to create his/her own song playlist and play it while playing or simulating the program.

Alloy Displayer

Alloy displayer is a java stand-alone program that will take an Alloy model file and a Alloy theme file as inputs, and run all the commands in the alloy model file. It will then display the results of all the commands in separate tabs while applying the theme file to each command before displaying the results. The project is finished yet. I am planning to add more features and error checking to the program in the future.

SchoolBud

SchoolBud is java stand-alone program to calculate the GPA and help with generating a schedule for your classes. Given the target grade, the GPA calculator will display the grade you need in remaining assignments to achieve the target grade in the course. You can also create custom rubrics for each class. The class scheduler allows you to set preferences like whether you want your classes at the begining of the day, or whether you want your classes spaced out, etc. and generate a list of schedules based on those preferences.

Bare-Metal OS

Project written in C. This project is a 16-bit bare metal operating system from scratch. It has a boot loader to boot from a floppy disk image, a set of system calls that are made using interrupts, a file system, the ability to execute a program from a file, a command line shell with minimum of necessary commands like directory, listing, type, copy, delete, execute, adn multiprocessing and basic memory management.
